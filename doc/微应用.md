# 注意事项

- 1. 路由不能用懒加载
- 2. hash 模式，路由 path 需要统一添加路由匹配前缀 /micro。如：path: '/micro/about'
- 3. history 模式，可以通过配置 base 添加路由匹配前缀 /micro。如：base: '/micro'
